# AI-nanny-
เว็บแอปพลิเคชันคู่มือการอาบน้ำเด็กแรกเกิดฉบับสมบูรณ์สำหรับพ่อแม่มือใหม่ พัฒนาด้วย HTML, Tailwind CSS และ Chart.js ภายในประกอบด้วยเช็คลิสต์อุปกรณ์ กราฟวิเคราะห์ความยากง่ายในการดูแลแต่ละจุด และฟีเจอร์พิเศษ "พี่เลี้ยง AI" (Powered by Gemini API) ที่คอยตอบคำถามและให้กำลังใจพ่อแม่แบบ Real-time
